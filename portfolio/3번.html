표정 연습 / 교정 애플리케이션

1. 프로젝트 개요 (Overview)

"감정 표현이 서툰 사용자들을 위해, 내 표정을 인식하고 연습할 수 있는 게이미피케이션 앱을 제작했습니다."

👥 팀 / 역할 5명 / 기여도 60%
(모델링, 전처리, 서버 구축)
🛠️ 기술 스택
PythonFlutterFlaskEfficientNetPostgreSQL
🔗 주요 특징 실시간 표정 인식
경량화 모델(Mobile/EfficientNet)

2. 배경 및 문제 정의 (Challenge)

🧐 개발 목표

표정 짓기를 어려워하는 사람들에게 '재미'와 '훈련'을 동시에 제공하는 앱을 기획했습니다. 한국인의 얼굴 특성과 제한된 감정(기쁨, 화남, 당황, 슬픔)을 정확히 분류하는 것이 핵심이었습니다.

🚧 기술적 난관 (Difficulties)

  • 데이터셋의 한계: 한국인 얼굴 데이터셋이 부족하고, 미세한 표정 변화를 잡아내기 어려움.
  • 모델 일반화 성능 저하: 훈련 셋에서는 높은 성능을 보이나, 실제 카메라 테스트 시 조명이나 각도에 따라 인식률이 급격히 떨어지는 과적합(Overfitting) 발생.

3. 해결 과정 (Process & Action)

🔬 모델 성능 비교 및 선정

모바일 환경 탑재를 위해 연산량(FLOPs) 대비 성능이 우수한 경량화 모델을 벤치마킹했습니다.

모델파라미터 수F1-Score정확도
MobileNet V24.2M0.8080.0%
ResNet5025.6M0.8281.7%
EfficientNet B05.3M0.8585.4%
ViT (Transformer)85M0.8685.7%

ViT가 가장 성능이 좋지만 파라미터 수가 너무 많아 모바일에 부적합하다고 판단, EfficientNet을 최종 베이스 모델로 선정했습니다.

🔧 트러블슈팅: 일반화 성능 확보

  1. 모델 복잡도 상향: EfficientNet B0에서 B3로 모델 체급을 키워 미세한 특징 학습량 증대.
  2. 데이터 증강(Augmentation): 회전, 밝기 조절, 노이즈 추가를 통해 다양한 실제 환경 모사.
  3. 분석: 증강 기법 적용 후 B3 모델에서 검증 손실(Validation Loss)이 안정화됨을 확인.
