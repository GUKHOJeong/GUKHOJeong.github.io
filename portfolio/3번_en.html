Facial Expression Practice Application

Project Topic

A facial expression practice application to help people who have difficulty expressing emotions by providing both fun and practice.


Project Challenges

The main challenges were working with a limited dataset: a specific race (Koreans) and only a few emotions (anger, joy, embarrassment, sadness).

Final Base Model: EfficientNetB0

This model was chosen as it offered the best performance while being a lightweight model.


Problem

Although performance was good, generalization performance was very poor. We initially suspected overfitting and proceeded with fine-tuning in stages, but the same issue persisted.

Solution

We adopted a paper's conclusion that B0 fails to classify properly with even a little complexity. After resetting to B2 for validation, performance improved slightly, but remained unstable.
