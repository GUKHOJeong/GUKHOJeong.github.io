Marine Fishery Catch Analysis Project

Project Topic

To identify the impact of marine climate factors on fishery catch and build a predictive model.


Project Challenges

1. Difficulty ensuring analysis reliability due to high volatility in catch data. An attempt was made to change the metric to 'catch rate,' but this did not fully resolve the volatility.


2. Significantly low prediction accuracy due to high RMSE/MAE metrics, despite applying various prediction models to complex data.


Project Information

  • Project Duration: 2025.04.30 ~ 2025.05.16
  • Team Size: 4 members
  • Personal Contribution: 70% (code writing, roadmap creation, hypothesis testing)
  • Related Skills: Deep Learning, Machine Learning, Time Series
  • Detailed Skills: seaborn, matplotlib, CNN, LSTM, XGB

Data Access

  1. Predictions using actual satellite imagery.
  2. Numerical data from satellite photos provided by marine sites.
  3. Numerical data provided by the Korea Meteorological Administration.
  4. Data Repository

Features Preprocessing

  1. Identifying time-series patterns using CNN.
  2. Domain-based feature selection.
  3. Dimensionality reduction based on PCA.
  4. Feature selection using the stepwise method.

The Problem (Very Low Performance)

The actual performance was very poor. The metric was changed from catch amount to catch rate, but performance remained low.

Solution

1. The actual catch amount and catch rate data showed poor performance due to instability.

Reason: Policy and artificial changes were made to fishing practices.

Solution (continued)

2. Based on a research paper, it was determined that if the number of data points is less than the number of parameters, overfitting and training issues can occur.

-> Use ML models (XGB, RF).

Final Insights

  • We concluded that changes in the marine environment do not have a significant impact on the catch amount and catch rate of various fish species.
  • Furthermore, we reached a final conclusion that machine learning models alone have limitations in predicting catch amounts due to complex, real-world data structures and numerous unpredictable non-linear factors, such as artificial intervention (fishing operations and policies).
